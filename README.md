## This is a very basic drawing program, similiar to Microsoft's Paint written in C language.
### What is used?  
#### -Code::Blocks IDE
#### -OpenGL graphics library (http://freeglut.sourceforge.net/)  
#### -DevIL image library (http://openil.sourceforge.net/)
